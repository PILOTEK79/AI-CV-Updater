# 🚀 AI CV Updater

An AI-powered Resume Analyzer built with **FastAPI, HTML, CSS, and JavaScript** that helps users evaluate and improve their resumes through automated scoring, strengths analysis, and personalized suggestions.

Try the project here:

**🔗 Live Website:** https://pilotek79.github.io/AI-CV-Updater/

* Upload a resume
* Receive an ATS-style score
* View strengths analysis
* Get improvement suggestions
* Experience a modern interactive UI

No installation required.

## 📌 Overview

AI CV Updater is a web-based application designed to provide quick feedback on resumes. Users can upload a resume file and instantly receive:

* ATS-style resume score
* Key strengths identified from the resume
* Personalized improvement suggestions
* Resume content preview
* Interactive and modern user interface
* Professional loading animations and visual effects

This project demonstrates full-stack development skills using Python for the backend and modern web technologies for the frontend.

---

## ✨ Features

### Resume Analysis

* Upload resume files
* Analyze resume content automatically
* Generate an overall resume score out of 100

### Strength Detection

The system identifies positive resume indicators such as:

* Python skills
* FastAPI experience
* JavaScript skills
* Project experience

### Improvement Suggestions

Provides actionable recommendations including:

* Responsive design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/PILOTEK79/AI-CV-Updater.git
cd AI-CV-Updater
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Backend

```bash
uvicorn main:app --reload
```

Backend will run on:

```text
http://127.0.0.1:8000
```

### Open Frontend

Simply open:

```text
index.html
```

or use Live Server in VS Code.
